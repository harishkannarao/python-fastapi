# python-fastapi
Repository to learn and explore API and Backend service with Python using FastAPI Framework

## Tools Required

* uv `0.8.15`
* make `3.81`
* git `latest`
* pycharm `latest`

## Commands

### Default command

    make

### Initialise/update dependencies
    
    make init_dependencies

### Run in development mode

    make fast_dev

### Run in production mode

    make fast_run
